## 🚀 반기별 대표 성과 (하이라이트)

### 🏆 증권투자권유자문인력 자격증 취득
- 증권투자권유자문인력 자격증 취득 (2024.10.31)

거시경제학과 금리, 채권에 대한 기초 개념 습득
유가증권, 코스닥 등 시장별 특징과 거래 방식 이해
파생상품인 ELS/ELB, 선물옵션 등의 특성과 거래 방식 이해
=> 주식, 채권과 파생상품에 대한 이해로 자산평가 업무에 대한 이해도 강화

### 🏆 대체거래소 주식 현재가 조회 기능 개발
- 대체거래소 주식 현재가 조회 기능 개발 (2024.11.01 ~ 2025.03.01)

- Before: KRX(한국거래소) 현재가 기준으로만 조회 가능
- After: 대체거래소와 KRX 현재가 모두 조회 가능하도록 기능 개발
실시간 조회 쿼리 최적화 및 Oracle 힌트 적용
기존에는 수동 or 종목코드 기준으로 정렬 기준을 변경해야 했으나 매입가 기준 종목 정렬 기능 개발로 고객 편의성 개선
시간대별 정규장, 단일가 시간 조회 기능 개발로 고객 편의성 개선
