## 🚀 반기별 대표 성과 (하이라이트)

### 🏆 해외주식 잔고 보정 집계 성능 개선
- Before: 40분 소요 → After: 2분 소요
- 방법: Nested Join을 Hash_AJ 힌트 기반으로 리라이트
- 결과: 운영 배치 효율 20배 향상, 야간 배치 안정화

### 🏆 잔고 변동 해석 쿼리 작성
- Before: 백오피스 수동 분석 15~20분
- After: SQL 한 번으로 5분 이내 분석
- 효과: 고객 응대 시간 75% 단축, 응대 정확도 향상

### 🏆 Oracle KEEP 문법 기반 대용량 집계 최적화
```sql
MAX(CASE WHEN 계좌거래일자 >= DATE '2024-01-01' THEN 1 ELSE 0 END)
  KEEP (DENSE_RANK LAST ORDER BY 계좌거래일자)
  OVER (PARTITION BY 고객번호)
```
- 핵심 원리: `KEEP`으로 집계 대상 행을 지정, `OVER`로 PARTITION 단위 집계 수행
- 성과: 180만 건 자료 출력 1시간 → 2분 단축
- 기술적 성장: Oracle 분석 함수 구조에 대한 심층 이해 확보
